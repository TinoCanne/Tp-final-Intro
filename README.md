La idea: 
Es una pagina en donde el usuario se crea su cuenta con informacion orientada a la musica. 
Dentro de la pagina el usuario va a poder filtrar por parametros una busqueda que le mostrara una sola carta sea de otro usuario, una banda o un lugar de encuentro. 
Si se quiere contactar, el usuario podra presionar un boton de Jam! y se mostrara la forma de contacto.
Se pueden crear bandas y el usuario puede ser parte de hasta 5 bandas distintas.
